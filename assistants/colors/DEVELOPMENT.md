## Scripts

| Script                 | Description                                                                                                        |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------ |
| `yarn test`            | Test the Assistant                                                                                                 |
| `yarn build`           | Builds the Assistant                                                                                               |
| `yarn package-tarball` | Builds the Assistant as a local `*.tgz` tarball suitable for installing via Sketch's _Install from Archive_ option |
